使用說明 - Instructions

點看詳細教學 - Click & Read Instructions
1. 貼上您想要縮短的網址並按下"縮短網址"按鈕，即可生成短網址。本站支援最長200字的原始網址。
2. 為提升安全性，本站提供的短網址不會直接跳轉目標網址，而是會先跳轉至本站頁面顯示對應的完整網址，再由使用者自行決定是否前往。
3. 在網路條件許可時，本站在創建短網址時，會同步使用Google Safe Browsing V4檢查原始網址，並在短網址被使用時的兩段式跳轉頁面顯示檢測結果，方便使用者快速評估原始網址的安全性，但請注意Google Safe Browsing V4的檢測結果僅是評估性質，不代表百分百準確。
4. 生成的短網址沒有使用效期，若碰上異常狀況包含但不限於硬體故障、資料庫異常、營運終止等不可抗力則不保證永久有效，請自行保管好原始網址。
5. 使用本服務時就表示已同意本站的所有使用條款。

1. Paste the URL you want to shorten and click the "Generate Short URL" button to create a short URL. This site supports URLs up to 200 characters long.
2. To enhance security, the short URLs provided by this site will not directly redirect to the target URL. Instead, they will first redirect to a page on this site displaying the full corresponding URL, allowing the user to decide whether to proceed.
3. When network conditions permit, this site will use Google Safe Browsing V4 to check the original URL while creating a shortened URL. The detection results will be displayed on the two-step redirect page when the shortened URL is used, allowing users to quickly assess the security of the original URL. However, please note that the Google Safe Browsing V4 detection results are for evaluation purposes only and do not guarantee 100% accuracy.
4. The generated short URLs do not have an expiration date. However, in cases of anomalies, including but not limited to hardware failure, database issues, or cessation of operations, the short URLs may not remain valid indefinitely. Please keep the original URL safe.
5. By using this service, you agree to all the terms of use of this site.
