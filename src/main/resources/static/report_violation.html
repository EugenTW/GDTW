關於濫用舉報和自動處理 - About Report Violation & Auto-Handling


1. 若您發現有濫用本網站功能之情形，請務必完整填寫回報表單進行舉報，以利站方進行後續處理。
2. 本網站設有自動舉報處理系統，將依據舉報次數進行階段性自動處理。為避免系統遭濫用，單一 IP 每三分鐘僅可針對同一類型服務提交一次舉報。
3. 自動化系統僅具輔助功能，若有特殊或嚴重情況，歡迎來信至站方信箱（gdtw2024@gmail.com），站方將於人力許可範圍內進行人工審核與處理。
4. 本站保留對自動處理及人工處理結果之最終解釋權與處理權限。

1. If you notice any misuse of this website's services, please complete the report form to notify the site administrator.
2. An automated reporting system is in place and will take action based on the number of reports. To prevent abuse, the system limits each IP to one report per service type every three minutes.
3. As the automated system has its limits, you may also report directly by email at gdtw2024@gmail.com. Manual review will be conducted subject to staff availability.
4. The website reserves the right to make final decisions regarding both automated and manual handling outcomes.
